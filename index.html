📧 Email Confirmation Setup
1
Create EmailJS account
Go to emailjs.com → Sign Up free → confirm email
2
Add Gmail as email service
Dashboard → Email Services → Add New → Gmail → connect your Gmail → copy the Service ID
3
Create email template
Dashboard → Email Templates → Create New → paste the template from the Template tab → Save → copy Template ID
4
Copy your Public Key
EmailJS → Account → General → copy your Public Key
Free plan: 200 emails/month. Upgrade at emailjs.com if needed.
